# Task 14 — Implement the web UI workbench

## Goal
Build the analysis-facing Web UI with required panels.

## Context
Read:
- `MASTER_SPEC.md`
- `docs/23_web_ui_master_spec.md`
- `docs/25_data_analysis_workbench_spec.md`
- `examples/ui_panels.example.yaml`

## Constraints
- the UI must read persisted artifacts;
- do not hide logic in the frontend that should live in reusable backend services;
- cross-model comparison and slot-sensitivity analysis are mandatory first-class panels.

## Done when
- the required pages exist;
- the UI can open completed runs;
- the UI can compare models and slot effects.
